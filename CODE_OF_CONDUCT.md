# RuyiRepo Issues | Code of Conduct

Before making any contributions to this repository, please ensure you have read and understood the following:

RuyiRepo is committed to being an open and inclusive open-source community. We welcome individuals or organizations to share any perspectives, suggestions, or opinions with RuyiRepo in a friendly manner. However, the following types of remarks are prohibited:

* Comments lacking basic respect toward RuyiRepo maintainers (e.g., "free-riders")
* Discourteous wording or tone
* Non-objective statements of fact
* Highly personal emotional or inflammatory language
* Content clearly not written by humans (e.g., generated by AI tools)
* Any content prohibited by national laws and regulations (specifically referring to China) or deemed inappropriate by RuyiRepo administrators

If any of the above rules are violated, depending on the severity, RuyiRepo administrators reserve the right to take the following actions:

* Close the corresponding issue
* Issue a formal warning to the involved GitHub user
* Temporarily suspend the GitHub user from the RuyiRepo organization
* Permanently ban the GitHub user from the RuyiRepo organization

Regardless of whether the above actions are taken, RuyiRepo may report the incident to GitHub or relevant authorities/organizations and publicly document it in [this issue](https://github.com/RuyiRepo/issues/issues/1).

## Appeal

If you believe you have been subjected to inappropriate handling, please send an email to support.ruyirepo AT isrc DOT iscas DOT ac DOT cn to explain the situation.
