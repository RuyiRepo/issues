# RuyiRepo Issues | 行为准则

在本仓库中进行任何贡献前，请确保已阅读并理解下列内容：

RuyiRepo 致力于成为开放包容的开源社区。我们欢迎任何人或组织以友善的方式向 RuyiRepo 提出任何观点、建议或意见。但以下言论是禁止的：

* 对 RuyiRepo 维护者缺少基本尊重的言论（“伸手党”）
* 不文明的措辞或语气
* 非客观的事实表述
* 严重个人情绪化或煽动性的语言
* 明显非人类书写（如使用 AI 工具生成）的内容
* 任何受国家法律法规禁止的，或 RuyiRepo 管理员认为不合适的内容

如果违反上述规则，根据情况严重性，RuyiRepo 管理员有权进行以下处理：

* 关闭相应 issue
* 向对应的 GitHub 用户提出正式警告
* 暂时将对应的 GitHub 用户从 RuyiRepo 组织中封禁
* 永久将对应的 GitHub 用户从 RuyiRepo 组织中封禁

无论是否进行上述处理，RuyiRepo 均可能向 GitHub 或有管辖权的机构/组织报告相应事件，并将其以公开形式记录在[此 issue](https://github.com/RuyiRepo/issues/issues/1)中。

## 申诉

如果您认为受到了不合适的处理，请向 support.ruyirepo AT isrc DOT iscas DOT ac DOT cn 发送邮件说明情况。
